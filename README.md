## API Configuration

1..The app uses the `OpenWeatherMap API`.
2..API Key: Replace the `apikey` variable in `Script.js` with your own key from `OpenWeatherMap`.
3..API URL: `https://api.openweathermap.org/data/2.5/weather?units=metric&q={city}&appid={apikey}`.
